Tickers are input in the format specified by Yahoo Finance.
For US assets, input the ticker as everything after the colon in a regular ticker. For example, 'NASDAQ:AAPL' is input as 'AAPL'.

For assets on international markets, input the ticker as everything after the colon, followed by a full stop and the exchange as a two character code indicated by Yahoo Finance. For example, 'ASX:BHP' is input as 'BHP.AX'.

For cryptocurrency, input the crypto ticker, followed by a hyphen and the conversion currency. For example, BTC to USD is input as 'BTC-USD'.
